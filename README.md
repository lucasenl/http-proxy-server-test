# http-proxy-server-test

#### 介绍
代理服务器测试

可以使用`DNS OVER TLS` 来解析域名

使用了`dns-over-tls`

https://www.npmjs.com/package/dns-over-tls

可以把http请求重定向到https



#### 软件架构
软件架构说明


#### 安装教程

1. nodejs v18.1.0
2.  Deno 1.21.1
3.  `yarn install`

#### 使用说明

1. `npm run serve`
2. `deno task serve`
3.设置代理服务器 地址 `127.0.0.1` 端口 `19001`
